This is nextjs,typescript and contentlayer example.

This example use:
- NextJS pages router + typescript
- Tailwind CSS
- Contentlayer (v0.3.1)
- Next-contentlayer (v0.3.1)

```bash
git clone https://github.com/fikriwebdev/next-contentlayer-example
```

then run

```bash
npm i
```

then run

```bash
npm run dev
```
